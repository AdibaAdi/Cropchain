# Cropchain Final Project

This project simulates a Waterfall software planning team using AutoGen and GPT-4o-mini.

## Team workflow
- Experiment 1: Team Lead
  - Customer Proxy
  - Project Manager
  - Requirements Engineer

- Experiment 2: Team Member 2
  - System Engineer
  - Software Engineer

- Experiment 3: Team Member 3
  - Test Engineer
  - Documentation Engineer
  - Comparative Analysis

## Shared setup
All team members must use:
- the same `.venv`
- the same `requirements.txt`
- the same `.env`
- the same model: `gpt-4o-mini`

## Output folders
- `src/outputs/experiment_1/`
- `src/outputs/experiment_2/`
- `src/outputs/experiment_3/`
